Moderation Commands

Command: Description: Usage:
?setup VERY IMPORTANT. Sets up the counting, mod and welome channel ?setup
setprefix [prefix] Sets the server prefix for this bot. Default is: ? ?setprefix !
?ban [@person] Bans a member ?ban @FusionSid
?kick [@person] Kicks a member ?kick @FusionSid
?giverole [@role] [@person] Gives a role to a person ?takerole @Admin @FusionSid
?takerole [@role] [@person] Takes a role from a person ?takerole @Moderator @FusionSid
?report [person/bug/message] Report a member, message or bug. Member and message reports get sent to the mod chat if you have one ?report message
?lockdown [channel=optional, defaults to the channel you type the command in] Lockdowns a channel (stops members from sending messages ?lockdown
?unlock [channel=optional, defaults to the channel you type the command in] Unlocks a channel thats on lockdown ?unlock #media
?reactrole [emoji] [@role] [message] Makes a message and reacting to the message will give you a role ?reactrole :emoji: @TestRole React to get role
?make_channel [name] Creates a channel ?make_channel memes
?make_vc [name] Creates a VC ?make_vc Hi
?clear [amount] Clears messages from the chat ?clear 10

?warn [@person] [reason] Warns a member ?warn @FusionSid
Custom VC This isnt a command but if you rename a VC to Custom VC, Joining it will create a new vc and then delete it upon leaving the vc

Ticket Commands

Command: Description: Usage:
?new [message=optional] This creates a new ticket. Add any words after the command if you'd like to send a message when we initially create your ticket ?new Help me please
?close Use this to close a ticket. This command only works in ticket channels. ?close
?addaccess [role_id] This can be used to give a specific role access to all tickets. This command can only be run if you have an admin-level role for this bot. ?addaccess 123456789
?delaccess [role_id] This can be used to remove a specific role's access to all tickets. This command can only be run if you have an admin-level role for this bot ?delaccess 123456789
?addadminrole [role_id] This command gives all users with a specific role access to the admin-level commands for the bot, such as ?addpingedrole and ?addaccess. This command can only be run by users who have administrator permissions for the entire server. ?addadminrole 123456789
?deladminrole [role_id] "This command removes access for all users with the specified role to the admin-level commands for the bot, such as ?addpingedrole and ?addaccess. This command can only be run by users who have administrator permissions for the entire server. ?deladminrole 123456789
